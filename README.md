# E-Class Dashboard

This project is a course dashboard web application built using Next.js. It provides a clean interface for viewing assignments and course information.

## Features

* Dashboard overview with course details
* Upcoming and completed assignments
* Recent announcements
* Deadline timeline
* Easy navigation links

## Tech Stack

* Next.js (App Router)
* React
* Tailwind CSS
* Radix UI components

## Getting Started

1. Clone the repository:
   git clone <https://github.com/GrapeGit/new_eclass_ui.git>

2. Navigate into the project folder:
   cd new_eclass_ui

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Open in browser:
   http://localhost:3000

## Project Structure

* `app/` – Main application routes
* `components/` – Reusable UI and dashboard components
* `lib/` – Mock data and utilities
